Giới thiệu dự án: Dự án sử dụng **Next.js** để hiển thị danh sách sản phẩm và trang chi tiết sản phẩm từ API.
Các tính năng chính:
1. Trang chủ: Hiển thị danh sách sản phẩm theo dạng lưới (Grid).
2. Trang chi tiết: Xem thông tin cụ thể của từng sản phẩm khi nhấn vào.
3. Responsive: Giao diện tự động căn chỉnh trên điện thoại và máy tính.

Cách chạy project:
1. Cài đặt thư viện: `npm install`
2. Chạy dự án: `npm run dev`
3. Truy cập: `http://localhost:3000`
